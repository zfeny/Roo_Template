# .roo_process 文档导航

- `REVIEW_GATE.md`：Review Gate 校验规则与证据规范
- `QUEUE_POLICY.md`：消息队列自动批准与风险边界
- `TODO_POLICY.md`：长任务 Todo 强制策略
- `WORKTREE_POLICY.md`：并行 WO 的 worktree 规范
- `TOOLS_POLICY.md`：自定义工具优先级、白名单与审计契约
- `AB_VALIDATION.md`：A/B 验证指标与记录模板

说明：
- 用户只通过 Roo 对话推进流程。
- 脚本与自动化由 agent 在会话内触发，不要求用户手动执行终端命令。
- 仓库级迁移/审计文档位于根目录：`MIGRATION_TOOLS_V3.md`、`PHASE0_ASSET_AUDIT.md`。
